Github+hexo搭建Pages

发表于 2017-04-09

平时写博客不多，一是自己也没太多时间，二是自己真的懒。但作为一个技术猿，没有一个自己的博客好像有点说不过去，于是决定还是要多总结多归纳。博客平台很多，之前也用过一些，比如csdn等，不过时间久了就废掉了，另外一个就是这些博客空间的主题样式自己很不喜欢，于是还是决定使用GitHub的pages，毕竟github会一直用下去，主题还能根据自己喜好定制，甚至全部自行开发。

hexo是一款基于Node.js的静态博客框架, 支持markdown语法，选择原因就不再细说了。

另外本文不是针对小白，仅会对关键信息做说明。
